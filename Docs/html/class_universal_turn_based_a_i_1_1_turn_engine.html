Detailed Description

The super-class for all Turn Engines. Implementations of this class control the search for the best Turn. Provides an entry point for Unity with GetNextTurn which can be used in a familiar coroutine pattern. Defines attributes common to all Turn Engines such as depth and time limits. Also provides the TurnReadyEvent which is triggered after a turn search has been completed and returns the best turn found.

System.Collections.IEnumerator UniversalTurnBasedAI.TurnEngine.GetNextTurn ( GameState  state)
inline

The entry point to the engine. Starts a new thread to run the search in and waits on it. Once the search is completed or timed out, calls the TurnReadyEvent to return the best turn found.

Typical usage from Unity is:

StartCoroutine(engine.GetNextTurn(state));
